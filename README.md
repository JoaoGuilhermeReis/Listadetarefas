# ✅ Lista de Tarefas — HTML, CSS e JavaScript

Uma aplicação simples e funcional de **Lista de Tarefas** desenvolvida com **HTML, CSS e JavaScript puro**, permitindo ao usuário **adicionar** e **remover tarefas** de forma prática e rápida, direto no navegador.

## 📌 Sobre o Projeto

O objetivo deste projeto é oferecer uma interface simples e intuitiva para gerenciamento de tarefas diárias.  
Ideal para iniciantes que estão aprendendo **manipulação do DOM**, **eventos** e **estilização com CSS**.

---

## ⚙️ Tecnologias Utilizadas
- **HTML5** — Estrutura da aplicação  
- **CSS3** — Estilização e layout  
- **JavaScript (ES6+)** — Lógica de adicionar e remover tarefas

---

## 🛠 Funcionalidades
- ➕ **Adicionar tarefa** digitando no campo de texto e clicando em "Adicionar"  
- 🗑 **Remover tarefa** individualmente ao clicar no botão de exclusão  
- 📋 Lista dinâmica atualizada em tempo real sem recarregar a página  
- 💾 (Opcional) Armazenamento no `localStorage` para manter tarefas após recarregar
